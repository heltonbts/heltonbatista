# Anthropic Cookbook Notes

Base copiada de `/tmp/claude-cookbooks` para servir como referência visual e estrutural neste projeto.

## Referências importadas

- `assets/references/baseline_saas.png`
- `assets/references/distilled_saas.png`

## Critério de escolha

Entre os materiais do repositório, a linha mais útil para uma landing page foi `images/frontend_aesthetics` com a variação SaaS. Ela traz uma direção visual mais próxima de página comercial do que os cookbooks focados em skills financeiras.

## O que foi reaproveitado

- contraste entre blocos claros e fundo com textura suave
- tipografia de impacto no hero
- CTA em destaque com secundário discreto
- blocos modulares para promessa, benefícios e prova

## O que ficou de fora

- exemplos financeiros e automações de documentos
- skills voltadas para Claude desktop
- notebooks e código Python sem relação direta com a landing page

## Como usar daqui

1. Trocar headline, subtítulo e CTA no `index.html`.
2. Ajustar paleta em `styles.css`.
3. Usar as imagens em `assets/references/` para manter coerência visual durante as próximas alterações.
